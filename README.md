# Full-Stack E-Commerce

A modern full-stack e-commerce web application built as a portfolio project.

## Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwind CSS

### Backend

- Express.js
- TypeScript

### Database

- PostgreSQL
- Prisma ORM

### DevOps

- Docker
- Docker Compose

## Planned Features

- User authentication
- Product catalog
- Shopping cart
- Checkout
- Admin dashboard
- Inventory management
- Order management
